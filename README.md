# ProductHub CRUD App

A simple full-stack Product CRUD application built using React Router, Express.js, and PostgreSQL.

This project was created for learning how frontend and backend technologies work together in a real-world CRUD application.

---

# 🚀 Tech Stack

## Frontend
- React
- React Router
- Tailwind CSS
- Axios
- Lucide React Icons

## Backend
- Node.js
- Express.js
- PostgreSQL
- pg
- CORS

---

# ✨ Features

- Add Products
- View All Products
- Update Products
- Delete Products
- REST API Integration
- Responsive UI
- Form Handling
- Dynamic Routing
- PostgreSQL Database Integration
